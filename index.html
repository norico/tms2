Welcome to TrackMania Serveurs Scripts GitHub Pages.

Trackmania Serveur Script a été écrit dans un premier temps pour l'administration de mon premier serveur dédié sur TMN. (https://github.com/norico/tms)

Vous souhaitez travailler sur une branche, ou tout simplement tester les scripts, allez-y, c'est fait pour cela.

Mais ca fait quoi au juste

Avec TMS2 on peut installer un nouveau serveur canyon avec les contrôleurs Xaseco2 ou fast4. Il suffit juste de remplir le fichier de configuration, et de mettre en place le fichier .sh qui permet de lancer les scripts.

Bien sur, il vous faudra un accès au shell pour utiliser les commandes.
